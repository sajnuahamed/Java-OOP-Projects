\# Java OOP Projects



A collection of Java projects built to practice and demonstrate Object-Oriented Programming (OOP) concepts. These projects focus on designing real-world applications using classes, objects, encapsulation, inheritance, polymorphism, and other core Java principles.



Unlike my \*\*Java Utility Suite\*\*, which focuses on Core Java fundamentals, this repository contains larger, object-oriented applications that emphasize clean design, code organization, and maintainability.
