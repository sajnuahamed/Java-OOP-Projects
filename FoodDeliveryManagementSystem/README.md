# 🍔 Food Delivery Management System

A console-based Java application that simulates a simple food delivery management system. The project demonstrates key Object-Oriented Programming (OOP) concepts by managing different types of orders, calculating bills, and updating order statuses.

## ✨ Features

* Manage multiple order types:

  * 🍔 Fast Food Orders
  * 🍽 Fine Dining Orders
  * 🛒 Grocery Orders
* Calculate the final bill based on order type
* Update and track order status
* Search orders using Order ID
* Menu-driven console interface

## 🧠 OOP Concepts Demonstrated

* Abstraction
* Inheritance
* Interfaces
* Runtime Polymorphism
* Encapsulation
* Method Overriding
* Constructors
* `toString()`
* Arrays of Objects
