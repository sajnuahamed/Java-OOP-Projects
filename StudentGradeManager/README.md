\# Student Grade Manager



A console-based Java application that demonstrates Object-Oriented Programming (OOP) by managing student records. The program allows users to enroll students, calculate grades based on marks, and search for students using their roll numbers.



\## Features



\* Creates and manages student objects

\* Stores multiple students using an array of objects

\* Calculates grades automatically based on marks

\* Searches for students by roll number

\* Displays complete student information

\* Uses encapsulation through private fields and public getter/setter methods

\* Organizes code into separate classes following OOP principles



\## Technologies Used



\* Java

\* Object-Oriented Programming (OOP)

\* Classes and Objects

\* Constructors

\* Encapsulation

\* Arrays of Objects

\* Methods

\* `Scanner` for user input
